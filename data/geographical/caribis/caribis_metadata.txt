  Description:
    Abstract: The Caribbean Island Nation data set is a shapefile polygon modeling the coastlines of the countries in the region.  Shapefile format provides compatibility with georeferenced raster imagery used in a Geographic Information System (GIS).  In addition, this data set provides a relational perspective view (s) in a GIS of the survey area surrounding the Puerto Rico trench.
    Purpose: To supplement the main featured mosaic image of the Puerto Rico trench survey area in a GIS.
